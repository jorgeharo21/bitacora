SEMANA 3

DEFINICIONES GENERALES:

Están los sitios web, que es un espacio digital que esta constituido por múltiples documentos. Por otro lado, están el dominio y hosting, donde el dominio es la dirección y nombre del sitio web y el hosting es el espacio donde se almacena el sitio web.

DEFINICIONES DE HTML:

Acá podemos encontrar etiquetas, donde están escritos con códigos, que puede leer una persona sin la necesidad de compilarlo. Por otro lado, se tiene que considerar su anatomía, que respetando el de cada etiqueta y elemento, se puede ejecutar una pagina web. También está el elemento animado, acá se pueden colocar elementos sobre otros elementos.

ESTRUCTURA DE HTML:

Se divide 2 secciones, cabecera y cuerpo, en la cabecera se define el HEAD, acá va la información no visible, luego está el cuerpo, en esta sección se incluye toda la información visible para los usuarios

DIFERENCIAS EN PAGINA WEB ESTÁTICA VS PAGINA DINAMICA:

La dinámica, contiene una aplicación dentro de la propia página web, ofreciendo mayor interacción por parte del usuario y la estática, se enfoca a mostrar información permanente donde el navegante se limita a tener esa información sin poder interactuare con la página web que ha visitado

Además, tenemos distintas paginas donde nos ayudan a conseguir dominio y hosting: namecheap, GoDaddy, HostGator y .pe

INTRODUCCION A SITIOS WEB EN GOOGLE SITES

Es una aplicación online de Google que permite la creación de sitios web de una manera fácil y sencilla, se puede colocar imágenes textos links y adjuntar archivos, también sirve para usar en emprendimientos de personas.

DESEÑAR LA PAGINA WEB

Para crear una cuenta se necesita el Gmail, luego ir a las aplicaciones de Google, buscamos Google sites y elegimos un diseño ya hecho.

PERSONALIZAR DISEÑOS EN GOOGLE SITES

Se puede poner diferentes cuadros de imágenes, de texto, etc. Para dar una información importante. Se puede agregar fotos, fondos, cambiar el estilo de letras, etc.

MENU DE NAVEGACION

Se puede elegir diseños predeterminados, y ya con ello se puede modificar a nuestro gusto. Por otro lado, se puede poner el indice y agregar un carrusel de imágenes, botones, línea divisora o insertar videos de youtube

SEMANA 5

DOCUMENTACION WEB

Nos ayuda a prevenir errores y nos orienta el correcto uso de la plataforma que se esté creando. Por otro lado, dentro de una estructura de documentación web, Se pueden encontrar imágenes, en esta sección es relevante encontrar evidencias de fotos, capturas de pantalla o videos. Además, están los enlaces, ya que estos nos ayudaran a permitir que las paginas se relacionen entre cada una de ellas.

Los enlaces tienen 2 partes, la primera es el enlace, donde puede picar un clic, el segundo es el destino, en este elemento puede ser una imagen, archivo o página web; El destino del enlace puede ser a un directorio archivo o elemento situado dentro de la pagina web, si fuera un directorio, se mostraría un archivo predeterminado, si fuera un archivo, este preguntara al usuario que desea hacer, si fuera un elemento el navegador abrirá la pagina web y se direccionara hasta el elemento enlazado

LISTAS ORDENADAS Y NO ORDENADAS

Son las que se encuentran ENTRE LAS ETIQUETAS ul /ul si se desea se quiera añadir un nuevo punto, se tiene que hacer dentro de las etiquetas li y /li y las listas ordenadas, se tiene que hacer dentro de las etiquetas ol y /ol
